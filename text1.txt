## 1. What is HTML?

HTML (HyperText Markup Language) is used to **create the structure and content of web pages**. Everything you see on a webpage‚Äîtext, headings, images, links‚Äîcomes from HTML.

> **Remember:** HTML is the skeleton of a website.

---

## 2. How HTML Works

HTML uses **elements**.

An element has:

* **Opening tag** ‚Üí `<p>`
* **Content** ‚Üí `Hello`
* **Closing tag** ‚Üí `</p>`

Example:

```html
<p>Hello</p>
```

---

## 3. Void Elements

Some elements **do not have closing tags and cannot contain content**. These are called **void elements**.

Examples:

```html
<img>
<br>
<hr>
<input>
```

You may also see:

```html
<img />
```

Both are valid in HTML5.

> **Key rule:** Void elements only have a start tag.

---

## 4. What are Attributes?

An **attribute** gives extra information about an element and controls **how it behaves or looks**.

**Syntax:**

```html
<element attribute="value"></element>
```

> **Easy idea:** Element = what it is, Attribute = how it works

---

## 5. Common Attributes

### a) Links ‚Äì `href` and `target`

```html
<a href="https://example.com" target="_blank">Visit</a>
```

* `href` ‚Üí where the link goes (required)
* `target="_blank"` ‚Üí opens in new tab

---

### b) Images ‚Äì `src` and `alt`

```html
<img src="cat.jpg" alt="A sleeping cat">
```

* `src` ‚Üí image location (required)
* `alt` ‚Üí text description (for accessibility)

---

## 6. Boolean Attributes

Boolean attributes **do not need values**.

If present ‚Üí **true**
If absent ‚Üí **false**

Examples:

```html
<input type="checkbox" checked>
<input type="text" disabled>
<input type="text" required>
```

Common boolean attributes:

* `checked`
* `disabled`
* `readonly`
* `required`

---

## 7. Input and the `type` Attribute

The `type` attribute tells the browser what kind of input it is.

```html
<input type="text">
<input type="checkbox">
<input type="email">
```

## 9. Accessibility (Important)

Accessibility means making websites usable for **everyone**, including people with disabilities.

Examples:

* `required` for forms
* `readonly` for inputs

## 12. The `<link>` Element (Connecting External Files)

### What is the `<link>` element?

The `<link>` element is used to **connect your HTML file to external resources** like:

* CSS stylesheets
* Fonts (Google Fonts)
* Icons (favicons)

It does **not display anything** on the page. It works behind the scenes.

---

### Linking an External CSS File

```html
<link rel="stylesheet" href="./styles.css">
```

* `rel="stylesheet"` ‚Üí tells browser this is a CSS file
* `href` ‚Üí location of the file
* `./` ‚Üí current folder

üìå Best practice: Keep HTML and CSS in separate files.

---

### Where Does `<link>` Go?

Always inside the `<head>` section:

```html
<head>
  <link rel="stylesheet" href="./styles.css">
</head>
```

---

### Google Fonts with `<link>`

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Playwrite+CU&display=swap" rel="stylesheet">
```

* `preconnect` ‚Üí speeds up loading
* `rel="stylesheet"` ‚Üí loads font styles

---

### Favicon

```html
<link rel="icon" href="favicon.ico">
```

A favicon is the small icon in the browser tab.

## 13. HTML Boilerplate (Basic Template)

### What is a Boilerplate?

An **HTML boilerplate** is a ready-made template that contains the basic structure every HTML page needs.

Think of it as the **foundation of a house**.

---

### Basic HTML Boilerplate

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
    <link rel="stylesheet" href="./styles.css">
  </head>
  <body>
  </body>
</html>
```

---

### Key Parts Explained

#### 1. `<!DOCTYPE html>`

Tells the browser this is **HTML5**.

#### 2. `<html lang="en">`

Wraps the whole page and sets the language.

#### 3. `<head>`

Contains:

* meta info
* title
* link to CSS

#### 4. `<body>`

Contains:

* text
* images
* headings
* links

## 14. UTF-8 Character Encoding
